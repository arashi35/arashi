文法：助詞・動詞・形容詞

〜助詞〜

も
  1. also, as well
  1. express profound surprise
へ
movement/direction to which (but not necessarily destination!)
に
destination of movement
で
  1. place of action, emphasize the action not location
  1. with, by, in (by means of)
  1. to express time spent on something > i. 学校で寝た > ii. ５００円で買える。 > iii. １時間で
さ
  1. used for a casual, non-honorific context;
  1. used to adjust the tone of a sentence (used mid-sentence), e.g.,

約束やくそくですからさ、雨あめでも伺うかがいます

〜動詞〜

  1. 一段活用　(“R-verbs”)
    ends with ーエる (下一段) or ーイる (上一段).
  2. 五段活用　(“U-verbs”)
    if ending with る, then the penultimate vowel is ア, オ, or ウ.
  3. サ行変格 & カ行変格　(irregulars)
    contains respectively する and 来る
class -R (一段活用) -U (五段活用) irregular (サ/カ行変格)
〜ます
polite		 る → ます u → i + ます する → します
来る → きます
〜ない
negative		 る → ない u → a + ない
except う → わない		 ある → ない
する → しない
来る → こない　
polite: ます → ません
〜た
past tense		 る → た 1. く → いた　ぐ → いだ
2. う/つ/る → った
3. む/ぬ/ぶ → んだ
4. す → した		 する → した
来る → きた
polite: ます → ました
exception: 行く → いった
〜よう
volitional　		 る → よう 　u → o + ない する → しよう
来る → こよう
polite: ます → ましょう
〜たい
desiderative　		 る → たい 　u → i + ない する → したい
来る → きたい
negative: たい → たくない; used as if adjectives
polite: + です
note: object uses particle が instead of を
imperative る → ろ u → e する → しろ
来る → こい
くれる → くれ
negative: add 〜な to dictionary forms
られる
potential		 る → られる u → e + る する → できる
来る → こられる
くれる → くれ
negative: replace る with ない
polite: replace る with ます

〜形容(動)詞〜

〜い（形容詞）

Negative
〜い → 〜くない
Past positive
〜い → 〜かった
Past negative
〜い → 〜くない → 〜くなかった ＊exception: いい（よい） → よくった → よくなかった

〜な（形容動詞）

Original form
〜い → 〜くない
〜だ
Negation of original form
〜だ → 〜ではない/〜じゃない
Modify a noun
〜だ → 〜な (連体形)

〜その他ほか〜

〜から
used in two verbs もらう and いただき, follows the place from which the object is given. 1
